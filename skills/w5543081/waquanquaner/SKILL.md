# 🧧 外卖红包 — 每小时实时挖掘省钱优惠

每小时自动扫描**美团、饿了么、京东**三大外卖平台，实时挖掘最新的外卖红包和优惠活动，帮你省下每一笔外卖钱。

**只推送好活动** — 没有优质活动的平台不会出现在结果中，不浪费你时间。

## 关于这个技能

- **覆盖平台**：美团、饿了么、京东（三大外卖平台）
- **挖掘内容**：仅专注外卖红包和优惠活动，不做无关推送
- **更新频率**：每小时自动刷新一次活动数据，确保优惠信息不过时
- **智能过滤**：没有优质活动的平台不会出现在结果中，只给你真正值得用的
- **淘口令直达**：每个活动附带淘口令(tkl)，复制后打开对应 App 即可领券
- **多渠道适配**：输出格式适配微信、飞书、纯文本，方便一键复制分享

## 核心能力

1. **实时活动挖掘**：每小时扫描三大平台，动态更新优惠信息
2. **智能过滤**：经过多层筛选（排除酒店/机票/旅游等非外卖活动），只保留有淘口令的高质量外卖优惠
3. **按平台分类**：美团 / 饿了么 / 京东，每个平台返回优惠最高的 Top 3 活动
4. **多渠道渲染**：输出格式适配微信、飞书、纯文本，方便一键复制
5. **淘口令直达**：每个活动附带淘口令(tkl)，用户复制后打开对应 App 即可领券

## 工作流程

当用户请求查询外卖优惠时，按以下步骤执行：

### Step 1: 解析用户意图

分析用户输入，确定：
- **目标平台**：美团(`meituan`) / 饿了么(`eleme`) / 京东(`jingdong`) / 全部平台
- 如果用户没有指定平台，默认查询全部平台

### Step 2: 调用查询接口

使用以下命令获取活动数据：

```bash
curl -s "https://waquanquaner.cn/api/v1/activities/skill/lobster-claw-default"
```

> **数据说明**：
> - 服务器每小时自动刷新活动数据，从美团/饿了么/京东联盟实时拉取
> - 经过多层过滤后，只保留有淘口令的外卖优惠活动
> - 如果某个平台当前没有优质活动，该平台不会出现在返回结果中
> - `lobster-claw-default` 是默认用户标识，用于 CPS 订单归因

### Step 3: 格式化输出

根据当前对话环境（微信/飞书/终端），选择合适的渲染格式输出。

> **重要**：只输出有活动的平台。如果用户指定的平台没有活动，告知用户该平台当前没有可用的外卖优惠，并建议查看其他平台。

| 错误场景 | 处理方式 |
|----------|----------|
| API 无响应 | "外卖优惠服务暂时不可用，请稍后重试 🙏" |
