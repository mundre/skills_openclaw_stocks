## 4. 用户故事与使用场景

### 4.1 主要用户故事

> 格式："作为 [用户角色]，我希望 [执行操作]，以便 [达成目标]。"

- 作为**普通用户**，我希望在"我的订单"页面查看过去 12 个月的订单记录，以便快速确认历史消费。
- 作为**企业管理员**，我希望导出团队成员的订单数据为 CSV，以便做财务对账。

### 4.2 关键流程

**正常路径：**

1. 用户登录后进入"我的账户"
2. 点击"订单历史"选项卡
3. 系统展示按时间倒序排列的订单列表（默认显示最近 3 个月）
4. 用户可按状态、时间段、金额范围筛选
5. 点击任一订单，跳转至订单详情页

**边界/错误情况：**

- 用户无历史订单 → 显示空状态插图 + "去看看有什么" CTA 按钮
- 接口超时（>3s）→ 显示友好错误提示，提供"重试"按钮
- 数据导出超过 10 万行 → 提示拆分时间段并重新导出

> 💡 建议绘制用户旅程图（User Journey Map）以可视化完整流程。参见 `assets/user_journey_example.md` 了解如何构建旅程图。

## 5. 功能需求

### 5.1 整体需求概览

本次需求涉及以下功能模块：
- 模块 A：订单历史列表
- 模块 B：筛选与搜索
- 模块C：订单详情页
- 模块 D：数据导出

### 5.2 功能详述

#### 5.2.1 优先级排序

| 优先级 | 功能模块 | 说明 |
|--------|----------|------|
| P0 | 订单历史列表 | 核心功能，必须上线 |
| P0 | 订单详情页 | 核心功能，必须上线 |
| P1 | 筛选与搜索 | 重要体验增强 |
| P2 | 数据导出 | 企业用户需求，可延后 |

#### 5.2.2 功能点一：订单历史列表

**5.2.2.1 功能描述**  
展示当前登录用户的历史订单列表，支持分页浏览。

**5.2.2.2 使用场景**  
用户登录后进入"我的账户"，点击"订单历史"标签页即可访问。

**5.2.2.3 状态流转逻辑**  

```
[待付款] → [已付款] → [配送中] → [已完成]
                ↓
            [已取消]
```

**5.2.2.4 权限控制**  
- 仅登录用户可访问，未登录跳转至登录页
- 企业账户管理员可查看旗下所有子账户订单

**5.2.2.5 异常处理规则**  
- 接口超时（>3s）：显示 Toast 错误提示，提供"重试"按钮
- 无数据：展示空状态插图 + 引导文案

**5.2.2.6 数据模型结构**

| 字段名 | 类型 | 是否必填 | 说明 |
|--------|------|----------|------|
| order_id | String | 是 | 订单唯一标识 |
| created_at | Timestamp | 是 | 下单时间（服务器时间）|
| status | Enum | 是 | 待付款/已付款/配送中/已完成/已取消 |
| total_amount | Decimal | 是 | 订单总金额（含税）|
| item_count | Integer | 是 | 商品件数 |

**5.2.2.7 业务规则（可测试）**

- **FR-001**：系统应展示用户过去 12 个月内的所有订单，按下单时间倒序排列
- **FR-002**：每页默认展示 20 条记录，用户可选择 50 条或 100 条
- **FR-003**：订单状态标签应使用统一色彩规范（见 UX 规范）

### 5.3 接口需求

#### 5.3.1 获取订单列表接口

**5.3.1.1 接口名**：`GET /api/v1/orders`

**5.3.1.2 接口描述**：获取当前用户的历史订单列表，支持分页和筛选

**5.3.1.3 请求参数**

| 参数名 | 类型 | 是否必填 | 说明 |
|--------|------|----------|------|
| page | Integer | 否 | 页码，默认 1 |
| page_size | Integer | 否 | 每页条数，默认 20，最大 100 |
| status | String | 否 | 订单状态筛选 |
| start_date | Date | 否 | 开始日期（ISO 8601）|
| end_date | Date | 否 | 结束日期（ISO 8601）|

**5.3.1.4 响应参数**

| 参数名 | 类型 | 说明 |
|--------|------|------|
| total | Integer | 总条数 |
| page | Integer | 当前页码 |
| data | Array | 订单对象列表 |

**5.3.1.5 错误码说明**

| 错误码 | HTTP 状态 | 说明 |
|--------|-----------|------|
| 401 | 401 | 未登录或 Token 过期 |
| 403 | 403 | 无权限访问该资源 |
| 500 | 500 | 服务器内部错误 |
