# YumFu - 多世界文字MUD 🌍⚔️🪄

**选择你的冒险** - 多宇宙文字RPG，AI生成配图

---

## 🌐 支持的世界

### 🇨🇳 中文世界（武侠）
- ⚔️ **笑傲江湖** - 剑客时代，华山派纷争
- 🗡️ **倚天屠龙记** - 倚天剑与屠龙刀 *(即将推出)*
- 📖 **射雕英雄传** - 郭靖黄蓉江南七怪 *(即将推出)*

### 🇬🇧 英文世界（奇幻）
- ⚡ **Harry Potter** - 霍格沃茨，四大学院，魔法决斗
- 🐱 **Warrior Cats** - 四大族群，野猫冒险
- 🗡️ **Lord of the Rings** - 中土世界，魔戒远征 *(即将推出)*
- 🐉 **Game of Thrones** - 维斯特洛，铁王座 *(即将推出)*

---

## ✨ 核心特性

- 🌍 **双语支持** - 中文/英文无缝切换
- 🎭 **多宇宙** - 武侠、哈利波特、猫武士等
- 🎨 **AI配图** - 每个场景自动生成风格化插画
- 🤝 **多人游戏** - PvP对决、组队（仅OpenClaw）
- 📖 **剧情丰富** - 原汁原味的类型文学写作
- 💾 **存档系统** - 多存档槽位，随时读档

---

## 🚀 快速开始

```bash
/yumfu start
```

**第一步：选择语言**
```
🌍 欢迎来到YumFu！

1. 中文
2. English

回复：/yumfu lang 1
```

**第二步：选择世界**
```
选择你的江湖：

1. ⚔️ 笑傲江湖（武侠）
2. ⚡ 哈利波特（魔法）
3. 🐱  猫武士（野猫冒险）

回复：/yumfu world 1
```

**第三步：开始冒险！**

## 📦 安装

```bash
clawhub install yumfu
```

**需求：**
- Python 3.x + `uv`
- `GEMINI_API_KEY`（可选，用于AI配图）

**纯文字模式（无需API key）：**
```bash
export YUMFU_NO_IMAGES=1
```
