## 4. Report Assembly Guidelines

| Audience | Include | Exclude |
|----------|---------|---------|
| CEO / Board | Executive only | All technical detail |
| VP Marketing | Executive + Marketing sections 1-4, 6 | Crawl stats, schema |
| Marketing Manager | Full Marketing template | Deep technical debugging |
| Engineering Lead | Technical only | Revenue, keyword details |
| Client (agency) | Executive + Marketing 1-3, 6 | Internal cost data |

### Data Freshness Requirements

| Report Type | Maximum Data Age |
|------------|-----------------|
| Executive monthly | 3 days after period end |
| Marketing weekly | 1 day after period end |
| Technical | Real-time where possible |
| Quarterly review | 5 days after quarter end |

### Delivery Checklist

- [ ] All data sources verified and dated
- [ ] Period-over-period comparisons included
- [ ] Action items are specific, assigned, and time-bound
- [ ] Report tailored to audience level
- [ ] Sources cited for external benchmarks
