# Gap Analysis Frameworks

Frameworks for systematic keyword gap identification, content format gap analysis, funnel stage gap mapping, and opportunity scoring.

## 1. Systematic Keyword Gap Methodology

### Step 1: Define Your Keyword Universe

| Data Source | What It Provides |
|------------|-----------------|
| Search console | Keywords driving impressions and clicks (last 6 months) |
| SEO tool | Full keyword profile positions 1-100 |
| Content audit | Topics covered by existing content |

### Step 3: Calculate Keyword Overlap

| Segment | Strategic Meaning | Action |
|---------|------------------|--------|
| A (Only you) | Your content moat | Protect and strengthen |
| B (Shared) | Competitive battleground | Improve rankings |
| C (Only them) | Your content gaps | Prioritize and create |
| D (No one) | Untapped market | Evaluate and pioneer |

### Step 4: Filter and Categorize Gaps

**Filters:**
- Minimum search volume >100/month
- Maximum KD within your competitive range
- Must relate to your product/service
- Must match content types you can create

**Gap Categories:**

| Category | Description |
|----------|-----------|
| Topic gaps | Entire topics you haven't covered |
| Depth gaps | You cover it but competitor goes deeper |
| Angle gaps | Same topic, different perspective/audience |
| Format gaps | Same keyword, different content format |
| Freshness gaps | Same topic, their content is more current |

### Step 5: Prioritize Gaps

Use the scoring model in Section 4.

## 3. Funnel Stage Gap Identification

### Step 1: Define Funnel Stages

| Stage | Content Need | Expected Formats | Typical Keywords |
|-------|-------------|-----------------|-----------------|
| Awareness | Educational content | Blog posts, infographics | "what is", "how to" |
| Interest | Deeper education | Guides, webinars | "guide to", "tutorial" |
| Consideration | Comparison/evaluation | Comparisons, reviews, case studies | "best", "vs", "review" |
| Intent | Decision-support | Demos, pricing, ROI calculators | "pricing", "demo" |
| Purchase | Conversion | Product pages, signup forms | "buy", "sign up" |
| Retention | Support/engagement | Help docs, tutorials | "[product] how to" |

**Gap Severity**: Critical = zero content; High = significantly less than competitors; Medium = somewhat less; Low = roughly on par.

### Step 3: Identify Drop-Off Points

| Transition | Metric | Drop-Off Signal | Content Gap Likely |
|-----------|--------|----------------|-------------------|
| Awareness -> Interest | Bounce rate | >70% bounce | Missing "next step" content |
| Interest -> Consideration | Pages/session | <2 pages | Missing comparison content |
| Consideration -> Intent | Demo requests | Low conversion | Missing trust content |
| Intent -> Purchase | Abandonment | High rate | Missing objection-handling |

## 4. Opportunity Scoring Model

### Multi-Factor Scoring (1-5 each)

| Factor | Weight | 1 (Low) | 3 (Medium) | 5 (High) |
|--------|--------|---------|-----------|---------|
| Search Demand | 25% | <100/mo | 500-2,000 | >5,000 |
| Competitive Density | 20% | All competitors cover it | 1-2 cover it | No one covers it |
| Business Relevance | 25% | Tangential | Related | Core to offering |
| Creation Effort | 15% | Requires new capabilities | Moderate effort | Quick to create |
| Conversion Potential | 15% | Top-funnel | Consideration | Decision/transactional |

**Gap Priority Score** = Sum(Weight x Score)

### Priority Tiers

| Tier | Score | Timeline |
|------|-------|----------|
| P0 -- Fill Immediately | 4.0-5.0 | This week |
| P1 -- Fill This Quarter | 3.0-3.9 | 1-3 months |
| P2 -- Plan for Future | 2.0-2.9 | 3-6 months |
| P3 -- Monitor | 1.0-1.9 | Track quarterly |

### Quick-Win Score

```
Quick Win = (Search Demand + Business Relevance) - (Creation Effort + Competitive Density)
```

4+ = strong quick win | 2-3 = moderate | 0-1 = not a quick win | Negative = avoid

## Framework Selection Guide

| Situation | Primary Framework | Secondary |
|-----------|------------------|-----------|
| Need more traffic | Keyword Gap | Calendar Integration |
| Competitors outrank everywhere | Keyword + Format Gap | Opportunity Scoring |
| Low conversion rate | Funnel Stage Gap | Format Gap |
| Don't know what to write | All three gap types | Scoring + Calendar |
| Limited resources | Opportunity Scoring (quick wins) | Keyword Gap (filtered) |
| Entering new market | Keyword Gap (comprehensive) | Format + Funnel Gap |
