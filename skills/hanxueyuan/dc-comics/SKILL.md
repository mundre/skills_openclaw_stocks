# DC Comics

美国著名漫画出版商，拥有超人、蝙蝠侠等经典超级英雄 IP。

## 公司介绍

**DC 漫画**（Detective Comics）是美国两大漫画巨头之一，成立于 1934 年，隶属于华纳兄弟探索公司。

### 基本信息

| 项目 | 详情 |
|------|------|
| **成立时间** | 1934 年 |
| **总部** | 美国纽约 |
| **母公司** | 华纳兄弟探索 |
| **竞争对手** | 漫威漫画 (Marvel) |

### 经典超级英雄

| 英雄 | 首次登场 | 特点 |
|------|----------|------|
| **超人** | 1938 | 外星来客，超能力 |
| **蝙蝠侠** | 1939 | 亿万富翁，科技装备 |
| **神奇女侠** | 1941 | 亚马逊公主 |
| **闪电侠** | 1940 | 超速度 |
| **海王** | 1941 | 海洋统治者 |
| **绿灯侠** | 1940 | 能量戒指 |
