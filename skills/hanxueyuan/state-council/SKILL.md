# 国务院技能

## 技能描述
中华人民共和国国务院是最高国家行政机关。本技能提供国务院相关信息、政策法规、政务服务等。

## 核心功能
- 📜 **行政法规**：国务院令、行政法规
- 📋 **政策文件**：国发文件、国办文件
- 🏛️ **机构设置**：组成部门、直属机构
- 📰 **政务动态**：国务院会议、领导活动
- 📊 **政策解读**：政策背景、实施要点
- 🔗 **相关链接**：部委网站、地方政府

## 使用场景
- 查询国务院政策文件
- 了解行政法规
- 获取政务信息
- 查看机构设置

## 注意事项
- 以国务院官方网站为准
- 政策文件请注意时效性
- 具体执行请咨询相关部门
