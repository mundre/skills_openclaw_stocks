# 英国航空 (British Airways)

英国旗舰航空公司。

## 公司介绍

**英国航空公司**成立于 1974 年，总部位于英国伦敦，是英国最大的航空公司。

### 基本信息

| 项目 | 详情 |
|------|------|
| **成立时间** | 1974 年 |
| **总部** | 英国伦敦希思罗机场 |
| **枢纽机场** | 伦敦希思罗、伦敦盖特威克 |
| **机队数量** | 270+ 架 |
| **目的地** | 180+ 城市 |
| **联盟** | 寰宇一家 (创始成员) |
| **母公司** | 国际航空集团 (IAG) |

### 舱位等级

| 舱位 | 介绍 |
|------|------|
| **头等舱** | 顶级奢华体验 |
| **商务舱 (Club World)** | 全平躺座椅 |
| **优选经济舱 (World Traveller Plus)** | 宽敞舒适 |
| **经济舱 (World Traveller)** | 标准经济舱 |

### 会员计划 - 执行俱乐部 (Executive Club)

| 等级 | 要求 | 权益 |
|------|------|------|
| **蓝卡 (Blue)** | 注册即享 | 积分累积 |
| **银卡 (Bronze)** | 300 航段/年 | 优先值机、额外行李 |
| **金卡 (Silver)** | 600 航段/年 | 贵宾室、优先登机 |
| **白金卡 (Gold Guest List)** | 邀请制 | 专属礼遇、私人助理 |

### 积分权益

- **Avios 积分**: 飞行、信用卡、合作伙伴
- **积分兑换**: 免费机票、升舱、酒店
- **家庭账户**: 积分共享
- **积分购买**: 直接购买积分
- **积分 + 现金**: 混合支付选项

### 中国航线

- **通航城市**: 北京、上海、香港
- **航班频率**: 每日多班
- **机型**: B787、A350、B777 等

### 特色服务

- **英航贵宾室**: 希思罗 T5 专属贵宾室
- **机上餐饮**: 英式美食体验
- **机上娱乐**: 丰富娱乐系统
- **高速 WiFi**: 机上网络连接
- **特殊餐食**: 多种特殊餐食选择
