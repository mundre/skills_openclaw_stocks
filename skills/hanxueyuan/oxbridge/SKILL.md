# 牛津剑桥技能

## 技能描述
Oxbridge（牛津剑桥）是英国最顶尖的两所大学——牛津大学和剑桥大学的合称，代表英国高等教育最高水平。本技能帮助用户了解牛津剑桥历史、学院制、申请信息。

## 核心功能
- 🎓 **大学介绍**：牛津大学、剑桥大学
- 🏛️ **学院制度**：各学院介绍、特色
- 📚 **学术项目**：本科、研究生、博士
- 📝 **申请信息**：入学要求、申请流程
- 💰 **学费信息**：本国/国际学生学费
- 🏰 **校园生活**：住宿、社团、传统

## 使用场景
- 查询牛津剑桥信息
- 了解学院制度
- 获取申请要求
- 了解校园传统

## 注意事项
- 申请需提前准备
- 学院制是特色
- 以官网信息为准
