### 2026-03-06 - 火山引擎API错误

**问题**: 火山引擎 API Key 无效导致 401 错误

**原因**: 硬编码了无效的 API Key

**教训**: 
- 不要在代码中硬编码敏感信息
- 移除不使用的模型配置

**修复**: 从配置中移除火山引擎相关代码
