# 亚马逊运营执行

## 核心目标

将消费者洞察和竞品分析转化为可执行的Listing方案，覆盖从VOC到落地优化的全流程。

## 用户输入

- 品类/产品信息
- 竞品ASIN（可选）
- 评论数据（可选）
- 具体需求（VOC/标题/五点/A+/视频/品牌/合规）

## 功能模块

### 1. VOC/评论分析

**分析维度**：
- 消费者画像（年龄/性别/收入/使用场景）
- 好评Top5 → Listing卖点
- 差评Top5 → 产品改进方向
- 未满足需求 → 差异化机会
- 购买动机 → 营销策略

**VOC痛点分级**：
| 等级 | 说明 |
|------|------|
| 致命痛点 | 直接导致退货/一星差评 |
| 体验痛点 | 不影响使用但降低满意度 |
| 期望落差 | 产品未达宣传效果 |

### 2. 竞品卖点拆解

**分析维度**：
- 核心USP提取
- 差异化要素对比
- 视觉呈现分析
- 广告策略还原
- 流量词反查

**竞品卖点对比表**：
| 卖点 | 竞品A | 竞品B | 竞品C | 出现频率 | 是否核心 |
|------|:---:|:---:|:---:|:---:|:---:|
| 防水 | ✅ | ✅ | ✅ | 100% | 🔴必须有 |
| 可折叠 | ✅ | ❌ | ✅ | 67% | 🟡加分项 |

### 3. 差异化机会

**差异化方向矩阵**：
| 维度 | 方案 | 成本影响 | 竞争壁垒 |
|------|------|:---:|:---:|
| 功能差异化 | 增加竞品缺失功能 | 中 | 高 |
| 外观差异化 | 独特设计/配色 | 低-中 | 中 |
| 包装差异化 | 礼盒装/环保包装 | 低 | 低 |
| 品质差异化 | 材质/工艺升级 | 中-高 | 高 |
| 价格差异化 | 空白价格带切入 | 视情况 | 低 |

### 4. Listing优化

#### 标题撰写（100-125字符）

```
品牌名 + 核心关键词 + 核心卖点 + 材质/颜色 + 尺寸 + 数量
```

#### 五点描述（5条）

**结构**：
- 前2条：转化核心（痛点+解决方案+数字参数）
- 后3条：埋词补充（场景词/属性词）

#### A+内容模块

推荐模块组合：
1. 品牌故事模块
2. 对比图模块
3. 功能特性模块
4. 产品细节模块
5. 使用场景模块

#### 视频脚本（30秒/分镜）

按3秒一个分镜输出脚本格式。

### 5. 合规检测

**检查维度**：
- 标题合规（字符/特殊字符/促销语）
- 图片合规（白底/像素/水印）
- 五点合规（违规词/字符数）
- 描述合规（HTML/外链）
- 后台关键词合规
- A+合规（促销/外链）
- 类目特殊限制

### 6. Rufus问答训练

**四类QA生成**：
| 类别 | 说明 |
|------|------|
| 买前关心 | 功能/兼容性/场景适配 |
| 买后预防 | 退货高频点预防性回答 |
| 竞品拦截 | 对比时提供弹药 |
| 关联场景 | 触发Cosmo场景推荐 |

### 7. 品牌规划

**品牌全案模块**：
- 品牌故事（使命/愿景/价值观）
- 三幕式叙事结构
- 视觉体系（Logo/色系/字体）
- 目标人群洞察
- 品牌调性定义

## 输出格式

### VOC分析报告

```
## 消费者画像
- 核心人群：XX-XX岁，XX性，年收入$XX-$XX
- 购买动机：XX > XX > XX

## 痛点分析
1. 【致命】XX问题（占比XX%）
2. 【体验】XX问题（占比XX%）
3. 【期望】XX未满足（提及率XX%）

## 机会点
- 差异化机会：XX（竞品未解决）
- 场景扩展：XX（使用场景延伸）
```

### Listing优化方案

针对每个模块（标题/五点/A+/视频）输出：
- 当前问题
- 优化方案
- 具体文案/设计要求
