# 运营执行Checklist

## VOC分析Checklist

```
□ 收集评论数据（至少100条）
□ 人口统计提取（年龄/性别/收入）
□ 购买身份分类（自用/送礼/专业）
□ 使用场景提取
□ 好评Top5提炼
□ 差评Top5归纳
□ 未满足需求挖掘
□ 购买动机排序
□ 痛点分级（致命/体验/期望）
□ 差异化机会输出
```

## 竞品分析Checklist

```
□ 选取3-5个直接竞品
□ 提取核心USP
□ 对比差异化要素
□ 分析视觉呈现
□ 反查流量词
□ 分析广告策略
□ 还原推广节奏
□ 输出竞品对比表
□ 输出可攻击弱点清单
```

## Listing优化Checklist

### 标题
```
□ 字符数100-125
□ 无特殊字符
□ 无促销语
□ 无品牌堆砌
□ 核心关键词前置
□ 卖点+参数+数量
□ 移动端可读
```

### 五点
```
□ 5条（每条一个主题）
□ 前2条转化核心
□ 后3条埋词补充
□ 数字参数清晰
□ 无违规词
□ 移动端友好
```

### A+
```
□ 模块数量≥5
□ 品牌故事
□ 对比图表
□ 功能拆解
□ 使用场景
□ 技术规格
□ 移动端适配
```

## 合规检测Checklist

```
□ 标题字符≤200（推荐≤150）
□ 无!$?{}等特殊字符
□ 无Best/Free Shipping等促销语
□ 主图白底RGB 255,255,255
□ 主图占比≥85%
□ 无水印/Logo/边框/促销标签
□ 五点3-5条，每条10-255字符
□ 描述≤2000字符
□ 无外链/联系方式
□ HTML仅允许<b><strong><em><p>
□ Search Terms无品牌/竞品名
□ A+无促销/外链/无根据声明
□ 类目特殊限制确认
```

## Rufus QA训练Checklist

```
□ 生成20+组QA
□ 覆盖功能/场景/对比/负面/长尾
□ 每组含Question+Answer+Confidence
□ 防御性写作（针对竞品弱点）
□ 口语化表达
□ 场景化答案
```
