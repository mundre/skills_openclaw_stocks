# 金手指设计专项规则

适用场景：

- 修仙、玄幻、升级流、系统流、爽文
- 主角出身平凡、资质普通、起点过低
- 需要设计主角专属优势，但又不想把逻辑写崩

## 1. 核心定义

金手指通常同时具备三个特征：

- 专属：只有主角或极少数人拥有
- 持续：不是一次性道具，而是能持续影响成长
- 不对等：给主角带来别人难以复制的优势

本质上，它是主角在世界中的特殊筹码。

## 2. 两大来源类型

### A. 世界观金手指

来源于世界本身，有世界内逻辑。

常见形式：

- 上古遗物
- 天地奇物
- 稀缺异宝
- 失落传承
- 特殊血脉容器

优点：

- 更容易自洽
- 可被争夺
- 能自然制造冲突
- 有利于后续 IP 延展

风险：

- 如果世界里类似宝物太多，会稀释主角优势

### B. 人物金手指

更贴近主角自身，常和灵魂、意识或唯一绑定系统相关。

常见形式：

- 签到系统
- 任务系统
- 抽奖系统
- 绑定 AI
- 天赋面板

优点：

- 爽感直接
- 成长效率高
- 叙事节奏快

风险：

- 最容易冲击世界底层规则
- 如果奖励无代价，会迅速打穿张力

## 3. 设计优先级

先回答：

1. 主角当前真正缺什么
2. 金手指解决的是哪一部分问题
3. 它不能解决什么
4. 它会带来什么代价、风险或追杀

好的金手指不是“万能钥匙”，而是“偏科但关键”的筹码。

## 4. 世界观金手指的设计原则

### 来源清楚

至少回答：

- 它从哪里来
- 为什么会流落到主角手里
- 为什么此前没有被更强者长期掌握

### 稀缺成立

至少满足一种：

- 数量极少
- 类型独特
- 启动条件苛刻
- 使用代价极高

### 可成为冲突焦点

最好让金手指本身也能引发：

- 争夺
- 觊觎
- 误判
- 身份暴露
- 阵营变化

## 5. 人物金手指的兼容原则

人物金手指不是不能用，而是更需要检查兼容性。

重点检查：

- 是否违反世界底层规则
- 是否无成本发放资源
- 是否让所有外部努力都失去意义

如果世界设定是“任何力量皆有代价”，那系统免费送顶级资源就会冲垮世界逻辑。

更稳的做法：

- 给系统本身来源
- 给奖励增加条件
- 给收益附带损耗、任务、冷却、风险

## 6. 四个关键判断

### 1. 它服务什么成长困境

主角普通，不代表一定要给金手指。

先看故事真正缺的是：

- 资源
- 指导
- 时间
- 认知
- 机会
- 生存筹码

### 2. 它有没有边界

边界可以是：

- 功能边界
- 使用次数
- 冷却时间
- 资源消耗
- 启动条件
- 只能辅助，不能代打

### 3. 它有没有代价

代价可以是：

- 暴露风险
- 身体负担
- 精神污染
- 寿命损耗
- 关系代价
- 道德代价

### 4. 它会不会引发新矛盾

最好的金手指会同时带来：

- 更快成长
- 更大危险

如果只有收益没有反噬，后期很容易空心化。

## 7. 叙事作用

金手指不只负责“爽”，还可以承担：

- 解释主角为何能起步
- 让主角与大势力产生联系
- 成为悬念入口
- 成为追杀和争夺的导火索
- 为升级体系提供独特路径

## 8. 高风险失败模式

- 凭空出现，没有来源
- 功能过全，什么都能解决
- 奖励无代价，世界规则被打穿
- 主角胜利全靠外挂代打
- 金手指只提升数值，不制造任何新问题
- 明明设定资源稀缺，却让主角轻松无限拿资源

## 9. 写前检查

每次使用或强化金手指前先问：

1. 这次它解决了什么具体问题
2. 这次收益是否仍符合世界规则
3. 这次使用会暴露什么、失去什么、引来什么
4. 没有它时主角是否仍需做出真实努力

## 10. 写后检查

- [ ] 金手指来源清楚
- [ ] 金手指不破坏底层世界规则
- [ ] 金手指有边界、限制或代价
- [ ] 金手指没有把外部冲突全部抹平
- [ ] 金手指反而引出了新的风险或争夺
- [ ] 主角的成长不只是外挂代打
