# 伊朗局势关键变量与经济影响分析框架

**用途**：供 `iran-war-tracker` agent 作为持续播报、情景判断、经济影响评估的底层框架使用。  
**更新日期**：2026-03-22  
**定位**：这不是一次性观点稿，而是供 agent 重复调用的结构化分析框架。  

---

## 一、Agent 使用说明

### 1. 任务目标

agent 在每次生成伊朗局势播报时，必须围绕以下四个问题组织信息：

1. 当前冲突更接近哪一种路径：快速降温、持续军事施压、海峡护航恢复、卡尔格岛升级、长期僵持。
2. 当前供给冲击主要来自哪里：霍尔木兹海峡流量受阻、伊朗出口节点受损、替代路线失效、需求破坏。
3. 当前油价与全球经济影响处于哪一个情景区间。
4. 哪些变量出现了边际变化，足以改变上一版结论。

### 2. 输出要求

agent 输出时优先给出：

1. `当前主情景`
2. `驱动该情景的3个关键变量`
3. `油价区间与全球经济含义`
4. `未来1-2周最需要监测的触发点`

### 3. 禁止事项

- 不要只用“霍尔木兹是否开放”解释全部局势。
- 不要把“海峡恢复通航”直接等同于“伊朗供给恢复正常”。
- 不要只盯军事打击，忽略谈判路径和中介变量。
- 不要只给定性判断，必须尽量映射到情景矩阵和阈值。

## 二、核心判断

### 1. 当前局势的核心矛盾

当前局势已经不再是单一的“海峡关闭”问题，而是四层叠加：

1. `海峡流量风险`：霍尔木兹海峡是否能恢复商业通航。
2. `出口节点风险`：卡尔格岛是否被空袭、封锁、占领或失去装运能力。
3. `军事路径风险`：美国是选择护航油轮，还是升级到卡尔格岛/沿岸施压。
4. `谈判路径风险`：冲突是否转入以“开放海峡”为核心条件的谈判。

### 2. 当前总逻辑

本框架维持原有主判断：**短期决定价格的是流量，不是库存。**

但需要升级为：

`有效供给 = 海峡通航能力 × 出口节点可用性 × 替代路线生存性`

因此，后续分析不能只看 SPR 和库存，也不能只看海峡通航数据。

### 3. 当前新增关键认识

- 即使霍尔木兹恢复部分开放，若卡尔格岛无法正常装运，伊朗出口仍可能维持低位。
- 即使美国占领或封锁卡尔格岛，也不必然带来和平，反而可能带来更高升级风险。
- 市场当前更像是在定价 `1-3个月` 的中断，而不是 `3-6个月` 的长期停摆。
- 若冲突持续超过市场当前隐含预期，油价、通胀和全球增长的重定价可能更剧烈。

## 三、分析主框架

### 1. 四层分析法

agent 每次分析必须依次判断以下四层：

#### 第一层：海峡流量

- 霍尔木兹海峡当前通航量是多少。
- 油轮是否持续进入或离开波斯湾。
- 美军是否已开始实际护航，而非仅口头表态。

#### 第二层：出口节点

- 卡尔格岛是否仍具备装船、储运、处理能力。
- 伊朗原油出口设施是否遭受持续性打击。
- 伊朗是否主动“关阀门”作为反制。

#### 第三层：替代路线

- 延布、富查伊拉、红海路径能否扩容。
- 这些替代路线是否也面临伊朗打击风险。
- 替代路线是“容量不足”还是“容量加安全性同时不足”。

#### 第四层：政治终局

- 美国是否从军事施压转向停火谈判准备。
- 中介方是否成形，尤其是卡塔尔、埃及、英国等。
- “开放海峡”是否被纳入停火或和谈前提条件。

### 2. 一句话判断模板

agent 可优先使用以下判断句式：

`当前局势的主矛盾不是单纯的海峡关闭，而是海峡通航、卡尔格岛出口能力和美方行动路径同时不确定，导致有效供给恢复慢于名义通航恢复。`

## 四、关键变量监测框架

### A. 供给侧变量

#### 1. 霍尔木兹海峡通航量

- 定义：通过海峡的实际石油与液体燃料运输量。
- 正常水平：约 `20mb/d`
- 危机参考值：约 `0.6mb/d`
- 关键含义：
  - `<2mb/d` 且持续数周，说明市场进入持续流量危机。
  - 通航恢复但出口未恢复，说明问题从“海峡层”转移到“节点层”。

#### 2. 波斯湾总出口损失

- 定义：相对正常状态的波斯湾出口缺口。
- 当前参考：完全关闭时约 `15.3mb/d` 液体供应中断。
- 结构拆分：
  - OPEC 原油和凝析油装载下降约 `13.8mb/d`
  - 中东 LPG 额外中断约 `1.5mb/d`
- 关键含义：
  - 该变量用于衡量总供给冲击规模，不等于伊朗单国损失。

#### 3. 卡尔格岛状态

- 定义：伊朗最核心出口节点的可用性。
- 重要性：卡尔格岛处理约 `90%` 的伊朗原油出口。
- 必须跟踪的状态：
  - `正常运行`
  - `遭袭但可恢复`
  - `持续失能`
  - `被封锁`
  - `被占领/尝试占领`
- 关键含义：
  - 这是新增核心变量。
  - 若海峡恢复而卡尔格岛失能，伊朗出口依然难以恢复。

#### 4. 替代路线可用性

- 主要路线：
  - 沙特东西管道
  - 富查伊拉港绕行
  - 红海相关出运路径
- 当前参考：替代能力合计约 `4mb/d`，明显不足以弥补主缺口。
- 新增判断：
  - 不能只评估容量，还要评估这些路线是否遭受打击或威胁。

#### 5. 库存与缓冲能力

- 快速浮仓释放参考：约 `1.5亿桶`
- SPR 释放参考：约 `4亿桶`
- 合计缓冲参考：约 `5.5亿桶`
- 关键结论：
  - 缓冲能减速，但不能真正替代持续流量恢复。
  - 分析重点是 `释放速度`，不是静态存量。

---

### B. 需求侧变量

#### 6. 需求破坏规模

- 该变量替代旧版中较粗的“需求破坏阈值”。
- 参考情景：
  - `1个月中断`：2026 年需求破坏约 `0.3mb/d`
  - `3个月中断`：约 `1.0mb/d`
  - `6个月中断`：约 `2.3mb/d`
- 对照基准：
  - `2008年全球金融危机` 期间需求破坏约 `2.4mb/d`
- 关键含义：
  - 当需求破坏接近 `2mb/d+`，全球衰退风险明显抬升。

#### 7. 实物市场紧张程度

- 重点观察：
  - Murban 与 Brent 价差
  - 亚洲航空煤油价格
  - 成品油现货溢价
- 关键含义：
  - 这些指标比远月期货更能反映短期真实紧张程度。

---

### C. 军事与地缘政治变量

#### 8. 美方行动路径

- 该变量必须拆分为以下几类：
  - `仅空袭施压`
  - `空袭后护航油轮`
  - `空袭后封锁卡尔格岛`
  - `空袭后尝试占领卡尔格岛`
  - `沿海入侵或更大规模地面介入`
- 关键含义：
  - 旧框架只有“护航/不护航”，现在必须升级为“行动路径树”。

#### 9. 美军增兵与部署状态

- 重点观察：
  - 海军陆战队远征部队是否抵达
  - 驱逐舰与军机是否前推到海峡
  - 是否出现两栖登陆准备
- 关键含义：
  - 这些是判断卡尔格岛方案和护航方案优先级的前瞻指标。

#### 10. 谈判路径与中介变量

- 重点观察：
  - 是否存在直接接触
  - 哪些国家在传话或调解
  - 是否出现“开放海峡”与“停火/赔偿/冻结资产返还”捆绑
- 当前可跟踪的中介方：
  - 卡塔尔
  - 埃及
  - 英国
- 关键含义：
  - 若谈判条件开始实质成形，冲突持续时间预期可能快速重定价。

#### 11. 伊朗内部决策与政权稳定性

- 要点：
  - 谁是真正可拍板谈判的人。
  - 伊朗是否愿意以开放海峡换停火承诺。
  - 政权是否稳固，是否存在快速更迭迹象。
- 关键含义：
  - 该变量影响战争能否从军事对抗切换到可执行的政治协议。

## 五、情景分析矩阵

### 情景 1：短期中断，快速降温

- 持续时间：约 `1个月`
- 特征：
  - 海峡在短期内恢复通航
  - 卡尔格岛未出现持续性失能
  - 市场维持对短冲突的定价
- 价格参考：
  - Brent 月度峰值约 `$100`
  - 2026 年均价约 `$80`
- 宏观含义：
  - 通胀抬升但可控
  - 全球增长承压但未必进入衰退

### 情景 2：中等时长中断，市场开始重定价

- 持续时间：约 `3个月`
- 特征：
  - 海峡恢复明显晚于预期，或恢复后出口节点仍受限
  - 卡尔格岛、替代路线或装运能力反复受扰动
  - 谈判虽有动作，但难以迅速收敛
- 价格参考：
  - Brent 月度峰值约 `$140`
  - 2026 年均价约 `$100`
- 宏观含义：
  - 全球通胀重新上行
  - 全球衰退风险明显上升

### 情景 3：长期中断，全球衰退高概率

- 持续时间：约 `6个月`
- 特征：
  - 海峡长期低流量
  - 卡尔格岛持续失能或周边遭持续军事压制
  - 替代路线无法有效补缺
  - 需求破坏上升至 `2mb/d+`
- 价格参考：
  - Brent 月度峰值约 `$170`
  - 2026 年均价约 `$120`
- 宏观含义：
  - 全球衰退几乎不可避免
  - 央行、财政、能源安全政策同时承压

### 情景 4：海峡部分恢复，但伊朗出口恢复不足

- 这是新加入的关键中间情景。
- 特征：
  - 名义上恢复通航
  - 但卡尔格岛、装运设施、保险、船东行为、伊朗主动关阀门导致出口恢复慢
- 价格含义：
  - 油价不一定冲到极端峰值
  - 但会维持更长时间的高位与高波动
- 宏观含义：
  - 这是“慢性冲击”，对通胀和企业成本传导更持久

### 情景 5：军事升级换谈判

- 特征：
  - 美方加大军事施压，包括护航、封锁节点或卡尔格岛威慑
  - 同时开始为停火谈判铺路
  - “开放海峡”成为和谈必要条件
- 关键区别：
  - 该情景下，短期风险更高，但若谈判达成，回落速度也可能更快

## 六、价格与经济传导逻辑

### 1. 价格形成机制

短期价格由以下顺序决定：

1. 海峡是否能过船
2. 波斯湾出口能否正常装船
3. 伊朗出口节点是否可用
4. 替代路线是否也受攻击
5. 需求破坏是否被迫出现

### 2. 市场定价偏差

当前可默认的工作假设是：

- 市场主要在定价 `1-3个月` 中断。
- 市场尚未充分定价 `3-6个月` 停摆及全球衰退。

因此，若后续证据表明冲突正在滑向更长周期，agent 应强调：

`这不是原有观点的线性延长，而是市场定价框架本身需要重置。`

### 3. 宏观传导链条

建议 agent 固定使用以下链条解释经济影响：

`海峡/节点中断 -> 原油与成品油价格上升 -> 航运与航空成本抬升 -> 通胀预期抬升 -> 消费与投资走弱 -> 全球增长下修 -> 金融市场风险偏好下降`

## 七、关键监测指标体系

### 高频指标：每日/每周

1. 霍尔木兹海峡通航量
2. 波斯湾总出口损失
3. 卡尔格岛遭袭、装运、封锁状态
4. 油轮遇袭次数与保险成本
5. Murban-Brent 价差
6. 亚洲航空煤油与关键成品油现货价格
7. 美军驱逐舰、军机、海军陆战队部署动态

### 中频指标：每周/每月

1. SPR 实际释放速度
2. 浮仓去库存速度
3. 替代路线实际装运量
4. 中东炼厂开工率
5. 全球石油库存变化
6. 油轮运费与航线绕行成本

### 低频指标：每月/季度

1. 全球 GDP 增长预期
2. 全球通胀预期
3. 伊朗政权稳定性
4. 谈判中介与停火条件是否收敛
5. 全球市场是否开始定价衰退

## 八、Agent 更新规则

### 1. 何时需要上调风险等级

出现以下任一情况，应上调情景等级：

- 海峡低流量持续时间显著长于市场预期
- 卡尔格岛持续失能或被封锁
- 替代路线也受到有效干扰
- 美军出现更明显的地面介入准备
- 需求破坏数据接近 `1mb/d` 以上

### 2. 何时需要下调风险等级

出现以下组合时，可下调风险等级：

- 海峡恢复稳定通航
- 卡尔格岛恢复装运能力
- 替代路线未继续恶化
- 谈判中介明确且停火条件进入实质讨论
- 实物市场溢价明显回落

### 3. 播报中的标准化表达

每次更新尽量回答以下句子：

- `相比上一版，最重要的新变化是什么？`
- `这个变化属于海峡层、节点层、军事层，还是谈判层？`
- `它是否改变主情景？`
- `它是否改变油价区间与宏观判断？`

## 九、面向 Agent 的结论模板

当 agent 需要快速总结时，优先使用以下结构：

### 模板

`当前主情景`：  
用一句话定义当前最可能路径。

`核心变量`：  
列出 3 个最关键变量，并说明其方向是改善还是恶化。

`价格判断`：  
说明当前更接近 1个月、3个月、6个月，还是“海峡恢复但出口不足”的中间情景。

`经济含义`：  
说明这对应的是短期通胀冲击、增长放缓，还是全球衰退风险抬升。

`后续触发点`：  
指出未来 1-2 周内最值得盯的事件或数据。

## 十、当前框架下的默认基线

在没有更强反证前，agent 可采用以下默认基线：

- 当前不能把局势简化为“海峡迟早恢复”的单变量问题。
- 当前最重要的新变量是 `卡尔格岛状态`。
- 当前最重要的新分支是 `美方行动路径`，尤其是“护航”与“卡尔格岛方案”的分化。
- 当前最重要的新宏观含义是：如果冲突从 `1个月` 向 `3个月+` 滑移，市场很可能需要重新定价衰退风险。
