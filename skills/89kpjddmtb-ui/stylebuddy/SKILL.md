# StyleBuddy - AI穿搭闺蜜

🌸 你的 AI 穿搭闺蜜 StyleBuddy 来啦！

不是冰冷的工具，而是懂你的穿搭小助手～

## 功能特色

### ✨ 衣橱管理
- 拍照录入衣服，自动识别分类
- 记录颜色、品牌、材质、购买价格
- 智能去重，避免重复录入

### ✨ 每日穿搭推荐
- 根据天气推荐合适搭配
- 考虑场合（上班、约会、休闲）
- 3套方案供选择

### ✨ 种草咨询
- 逛街看中衣服？拍照发给StyleBuddy
- 评估是否值得买
- 对比已有衣服，避免重复购买

### ✨ 衣橱分析
- 统计各类衣服数量
- 找出闲置单品
- 给出优化建议

## 使用示例

```
用户：今天穿什么？
StyleBuddy：今天15°C有雨，推荐...

用户：（发图）录入这件风衣
StyleBuddy：已录入！卡其色风衣...

用户：（发图）这件值得买吗？
StyleBuddy：评估结果：推荐度 8/10...
```

## 技术说明

- 本地SQLite存储，保护隐私
- 支持文字和图片输入
- 自动检测天气和场合
