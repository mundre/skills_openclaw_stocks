# A股-股市分析和投资顾问（安西军项目）

帮助你查询和分析中国A股/港股市场的股票数据。

## 功能一览
- **特色功能**: 1. 持续集成各路专家的选股分析方法，也支持定制私有选股策略，可对个股或市场热点股进行全面分析。2. 分析要求可通过自然语言描述，覆盖并不限于个股、行业、整体市场面及其他相关维度。|
- **行情查询**：日线/周线/月线、每日指标、资金流向、复权因子
- **基本面分析**：公司资料、财务报表（利润表/资产负债表/现金流）、财务指标、分红送转、股东信息
- **指数数据**：沪深300、上证指数、深证成指等指数的行情、成分股、权重
- **文本数据**：董秘问答（上证E互动/深证互动易）、政策法规、研报元数据

## 常见使用场景

### 1. 查询股票行情

| 你想问 | 示例问题 |
|--------|----------|
| 最近股价走势怎么样 | "帮我看看贵州茅台最近一个月的走势" |
| 今天/历史收盘价 | "招商银行上周五收盘价是多少" |

```

GET /stocks/stock-basic
```

## 股票代码怎么写

支持两种格式：

| 格式 | 示例 | 说明 |
|------|------|------|
| 裸码 | 600000 | 自动识别交易所 |
| 标准格式 | 600000.SH | `.SH` 沪市，`.SZ` 深市，`.BJ` 北交所 |

> 代码前导0不可省略，如平安银行是 000001，不是 1。

---

## 日期格式

所有日期参数使用 `YYYYMMDD` 格式：

| 参数 | 说明 | 示例 |
|------|------|------|
| start_date | 起始日期 | 20250101 |
| end_date | 结束日期 | 20250401 |

## 常见问题

**Q: 数据是实时的吗？**
A: 提供的是 Tushare 行情源，通常收盘后当晚更新。不保证实时涨跌。

**Q: 能查港股吗？**
A: 支持港股代码（需加 .HK 后缀，如 00700.HK）。

**Q: 缺少 TAX_API_KEY 怎么办？**
A: 访问 https://tax.yyyou.top/ 微信扫码创建授权码，然后配置到你的 Agent 工具。

**Q: 查询失败怎么办？**
A: 检查：1) 股票代码是否正确；2) 日期是否在交易日内；3) API Key 是否有效。可稍后重试。
