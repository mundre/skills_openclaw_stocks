# DailyLog
Structured daily reflection. Record wins, challenges, and learnings every day.
## Commands
- `log <category> <text>` — Log entry (categories: win/challenge/learning/note)
- `today` — View today's log
- `yesterday` — View yesterday's log
- `week` — Weekly summary
- `search <keyword>` — Search all logs
- `stats` — Logging statistics
- `export` — Export as markdown
## Usage Examples
```bash
dailylog log win "Shipped new feature ahead of schedule"
dailylog log challenge "Debugging took longer than expected"
dailylog log learning "Learned about Redis caching patterns"
dailylog today
dailylog week
```
