这是一个专业的植物识别与养护技能，能够帮助用户：

## 常见问题

### Q：如何判断植物是否缺水？

A：使用手指测试法 - 将手指插入土壤 2-3 厘米，感觉干燥即可浇水。也可以观察土壤颜色，发白变浅时需要浇水。

### Q：植物叶子发黄是什么原因？

A：可能原因包括：浇水过多或过少、光照不足、缺肥、土壤碱化、病虫害等。需要结合具体情况分析。

### Q：多肉植物怎么浇水？

A：多肉植物耐旱怕涝，应遵循"宁干勿湿"原则。一般每周浇水一次即可，冬季可每月浇水一次。

### Q：室内植物需要施肥吗？

A：需要。植物在生长期需要养分补充。一般春夏季每 2 周施一次薄肥，冬季停止施肥。

## 注意事项

1. 本技能提供的信息仅供参考，具体养护方法需根据实际情况调整
2. 如遇到严重病虫害或植物濒临死亡，建议寻求专业园艺师帮助
3. 部分植物有毒，请勿误食，放置在儿童和宠物接触不到的地方
4. 外来入侵物种请勿种植或传播
