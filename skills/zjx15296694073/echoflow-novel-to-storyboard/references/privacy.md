# Data Handling & Privacy | 数据处理与隐私

## 本地处理 | Local Processing

This skill processes all content using the **current conversation model** — no external API call is made.

- Novel text is processed within the conversation context.
- Screenshots extracted via browser tool: only the **extracted text** is used; the image file is **not** uploaded to any external service.
