# 梅花易数排盘技能 v3.0.0

> "不动不占，无事不占；不诚不占，不疑不占。" —— 邵雍《梅花易数》

## 1. Description

梅花易数是宋代邵雍（邵康节）所创的占卜方法，以"心易"为核心，起卦简便，断卦精准。

**核心功能：**
- ✅ 数字起卦（先天数）
- ✅ 时间起卦（年月日时）
- ✅ 方位起卦（八方）
- ✅ 随机起卦
- ✅ 互卦计算
- ✅ 变卦计算
- ✅ 体用分析（体卦/用卦/生克关系）
- ✅ 自动化断卦
- ✅ **v2.0.0 新增：**
  - 外应断卦（颜色/物品/人物）
  - 卦例库（经典卦例参考）
  - 吉凶量化评分（0-100）
- ✅ **v3.0.0 新增：**
  - **应期推算**（卦数/五行/节气综合推断）
  - **卦例记录功能**（自动保存占例到本地库）
  - **多爻动支持**（支持 1-3 个动爻的复杂卦象）
  - **断语细化**（按问事类型输出针对性建议）
  - **卦气旺衰**（结合月令/日辰判断卦气强弱）

**起卦原则：**
| 原则 | 说明 |
|------|------|
| 不动不占 | 无事不问，不随意起卦 |
| 不诚不占 | 心诚则灵，心不诚不占 |
| 不疑不占 | 有疑惑才占，无疑不占 |
| 一事一占 | 一卦只问一事，不重复占 |

## 3. How to use

### 调用方式

```bash
# 数字起卦（至少 2 个数字）
python3 ~/.openclaw/skills/meihua-yishu-skill/scripts/meihua_pan.py --numbers "123,456,789"

# 时间起卦
python3 ~/.openclaw/skills/meihua-yishu-skill/scripts/meihua_pan.py --date "2026-03-21 15:00"

# 方位起卦
python3 ~/.openclaw/skills/meihua-yishu-skill/scripts/meihua_pan.py --fangwei "东"

# 随机起卦
python3 ~/.openclaw/skills/meihua-yishu-skill/scripts/meihua_pan.py --random

# 问事断卦（支持：财运/事业/婚姻/健康/考试/投资/出行/诉讼/失物/其他）
python3 ~/.openclaw/skills/meihua-yishu-skill/scripts/meihua_pan.py --numbers "123,456" --question "财运"

# 输出 JSON 格式
python3 ~/.openclaw/skills/meihua-yishu-skill/scripts/meihua_pan.py --numbers "123,456" --json

# 保存卦例到本地库
python3 ~/.openclaw/skills/meihua-yishu-skill/scripts/meihua_pan.py --numbers "123,456" --question "财运" --save

# 查看历史卦例
python3 ~/.openclaw/skills/meihua-yishu-skill/scripts/meihua_pan.py --history --limit 10

# 搜索卦例（按问事类型）
python3 ~/.openclaw/skills/meihua-yishu-skill/scripts/meihua_pan.py --search "财运"
```

### 命令行参数

| 参数 | 简写 | 说明 |
|------|------|------|
| --numbers | -n | 数字起卦 (逗号分隔，至少 2 个) |
| --date | -d | 时间起卦 (YYYY-MM-DD HH:MM) |
| --fangwei | -f | 方位起卦 (东/南/西/北/东南/西南/西北/东北) |
| --question | -q | 问事类型 (财运/事业/婚姻/健康/考试/投资/出行/诉讼/失物/其他) |
| --random | -r | 随机起卦 |
| --json | -j | 输出 JSON 格式 |
| --save | -s | 保存卦例到本地库 |
| --history | -H | 查看历史卦例 |
| --search | -S | 搜索卦例 |
| --limit | -l | 历史/搜索结果数量限制 (默认 10) |

## 5. 梅花易数基础知识

### 八卦先天数

| 卦名 | 先天数 | 五行 | 象 |
|------|--------|------|-----|
| 乾 ☰ | 1 | 金 | 天 |
| 兑 ☱ | 2 | 金 | 泽 |
| 离 ☲ | 3 | 火 | 火 |
| 震 ☳ | 4 | 木 | 雷 |
| 巽 ☴ | 5 | 木 | 风 |
| 坎 ☵ | 6 | 水 | 水 |
| 艮 ☶ | 7 | 土 | 山 |
| 坤 ☷ | 8 | 土 | 地 |

### 体用关系

| 关系 | 说明 | 吉凶 |
|------|------|------|
| 用生体 | 用卦生体卦 | 大吉 |
| 体克用 | 体卦克用卦 | 吉 |
| 比和 | 体用同五行 | 吉 |
| 体生用 | 体卦生用卦 | 凶 |
| 用克体 | 用卦克体卦 | 凶 |

### 起卦方法

**1. 数字起卦**
- 第一个数 ÷ 8 余数 = 上卦
- 第二个数 ÷ 8 余数 = 下卦
- 第三个数 ÷ 6 余数 = 动爻

**2. 时间起卦**
- (年 + 月 + 日) ÷ 8 余数 = 上卦
- (年 + 月 + 日 + 时) ÷ 8 余数 = 下卦
- (年 + 月 + 日 + 时) ÷ 6 余数 = 动爻

**3. 方位起卦**
- 东方 = 震卦 (4)
- 东南 = 巽卦 (5)
- 南方 = 离卦 (3)
- 西南 = 坤卦 (8)
- 西方 = 兑卦 (2)
- 西北 = 乾卦 (1)
- 北方 = 坎卦 (6)
- 东北 = 艮卦 (7)

### 应期推算方法 v3.0.0

| 方法 | 说明 | 示例 |
|------|------|------|
| 卦数应期 | 上卦数 + 下卦数 | 乾 1+ 兑 2 = 3 日/周/月 |
| 五行应期 | 体卦五行旺时 | 金旺申酉，应申酉日/月 |
| 动爻应期 | 动爻数 | 三爻动，3 日/3 周 |
| 节气应期 | 结合节气判断 | 春占木旺，应寅卯日 |

**事业：**
- 用生体：贵人提拔，晋升有望
- 体克用：主动争取，可成
- 体生用：付出多，回报少
- 用克体：工作压力大，防小人

**婚姻：**
- 用生体：对方主动，感情顺利
- 体克用：需主动追求
- 体生用：付出多，易受伤
- 用克体：感情受阻，防第三者

**健康：**
- 用生体：身体康复快
- 体克用：需积极治疗
- 体生用：体质虚弱，需调养
- 用克体：病情较重，及时就医
