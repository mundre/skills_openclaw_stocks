# Biogas — Biogas Technology Reference

Quick-reference skill for biogas production, anaerobic digestion processes, and plant design.

## When to Use

- Designing or sizing an anaerobic digester
- Selecting and mixing feedstocks for optimal C:N ratio
- Troubleshooting digester performance issues
- Evaluating biogas upgrading to biomethane
- Calculating methane yield from organic waste

## Commands

### `intro`

```bash
scripts/script.sh intro
```

Overview of biogas — production process, composition, and applications.

### `feedstocks`

```bash
scripts/script.sh feedstocks
```

Feedstock types, methane potential, and C:N ratios for common substrates.

### `digester`

```bash
scripts/script.sh digester
```

Digester types: CSTR, plug-flow, batch, UASB — design parameters and selection.

### `process`

```bash
scripts/script.sh process
```

Four stages of anaerobic digestion: hydrolysis, acidogenesis, acetogenesis, methanogenesis.

### `parameters`

```bash
scripts/script.sh parameters
```

Critical operating parameters: temperature, pH, OLR, HRT, VFA/alkalinity.

### `upgrading`

```bash
scripts/script.sh upgrading
```

Biogas upgrading technologies: PSA, water scrubbing, membrane, amine scrubbing.

### `troubleshoot`

```bash
scripts/script.sh troubleshoot
```

Common digester problems: foaming, acidification, low yield — causes and fixes.

### `economics`

```bash
scripts/script.sh economics
```

Biogas plant economics: CAPEX, OPEX, revenue streams, payback calculations.

### `help`

```bash
scripts/script.sh help
```

### `version`

```bash
scripts/script.sh version
```
