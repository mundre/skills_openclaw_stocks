# 更新日志

- ✅ 初始版本发布

- ✅ Tavily API 深度搜索（8 个关键词组合）

- ✅ 标准化调查报告输出

- ✅ 年金类型精确判断

### 策略来源
基于 2026-03-19 深圳市道桥维修中心企业年金调查实战经验总结，结合 2026-03-20 腾讯公司企业年金搜索验证。

### 核心原则
1. 拿到数据必须自己解析 — 不依赖用户反馈
2. 搜索必须深入 — 前 20 条链接全部访问
3. 证据必须交叉验证 — 单一来源不作为高置信证据
4. 结论必须稳健 — 找不到就说"公开信息暂不足以确认"
5. 名称变更必须说明 — 改名/子公司/集团/简称/改制必须说明原因

### 工具使用顺序
1. **Tavily API 第一** — 8 个关键词组合全部搜索
2. **Multi Search Engine 第二** — 百度/Bing/360/搜狗 4 个引擎必须全部执行

### 技术栈
- Bash 5.0+
- curl
- jq
- Tavily API（推荐）
- pypdf（可选，PDF 解析）
