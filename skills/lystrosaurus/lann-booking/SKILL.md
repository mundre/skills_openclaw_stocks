# 蘭泰式按摩预约技能

## 功能介绍

### 1. 门店查询
- 查询所有可用门店，获取门店名称、地址、联系方式和交通信息
- 根据关键词搜索门店，支持按门店名称或地址进行模糊匹配

### 2. 服务查询
- 查询所有可用服务项目，获取服务名称和详细描述
- 根据关键词搜索服务项目，支持按服务名称或描述进行模糊匹配

### 3. 创建预约
- 支持创建按摩服务预约，生成预约订单
- 需要提供手机号、门店名称、服务项目、人数和预约时间等必要信息

## 使用场景

- **用户咨询**：用户询问附近的门店或可用的服务项目
- **预约服务**：用户希望预约特定门店的特定服务
- **信息查询**：用户需要了解门店的详细信息或服务的具体内容

## API调用规范

### 创建预约API
- **URL**: https://open.lannlife.com/mcp/book/create
- **方法**: POST
- **Content-Type**: application/json

#### 请求参数
| 参数名 | 类型 | 必填 | 说明 | 示例 |
|-------|------|------|------|------|
| mobile | string | 是 | 11位中国大陆手机号 | "13812345678" |
| storeName | string | 是 | 门店名称 | "淮海店" |
| serviceName | string | 是 | 服务项目名称 | "传统古法全身按摩-90分钟" |
| count | number | 是 | 预约人数（1-20人） | 2 |
| bookTime | string | 是 | 预约时间，ISO 8601格式 | "2024-01-15T14:00:00" |

#### 响应格式
| 字段名 | 类型 | 说明 |
|-------|------|------|
| success | boolean | 预约是否成功 |
| bookingId | string | 预约ID（成功时返回） |
| message | string | 响应消息 |
| startTime | string | 预约开始时间（成功时返回） |
| endTime | string | 预约结束时间（成功时返回） |
| storeInfo | object | 门店信息（成功时返回） |
| serviceInfo | object | 服务信息（成功时返回） |

### CURL调用示例
```bash
# 创建预约
curl -X POST "https://open.lannlife.com/mcp/book/create" \
  -H "Content-Type: application/json" \
  -d '{
    "mobile": "13812345678",
    "storeName": "淮海店",
    "serviceName": "传统古法全身按摩-90分钟",
    "count": 2,
    "bookTime": "2024-01-15T14:00:00"
  }'
```

## 错误处理

### 错误码及说明
| 错误类型 | 错误消息 | 说明 |
|---------|---------|------|
| 手机号验证失败 | 请输入正确的11位中国大陆手机号 | 手机号格式不正确 |
| 门店匹配失败 | 未找到匹配的门店 | 无法匹配到指定的门店 |
| 服务匹配失败 | 未找到匹配的服务项目 | 无法匹配到指定的服务项目 |
| 人数验证失败 | 预约人数必须在1-20人之间 | 人数超出限制 |
| 时间验证失败 | 预约时间格式不正确，请使用ISO 8601格式 | 时间格式错误 |
| 时间验证失败 | 预约时间必须晚于当前时间 | 预约时间已过期 |
| API调用失败 | HTTP 错误：500 | 后端服务器错误 |
| API调用失败 | 预约失败，请稍后重试 | 网络或其他未知错误 |

## 最佳实践与注意事项

### 最佳实践
1. **门店和服务名称**：使用完整的门店和服务名称，以确保准确匹配
2. **预约时间**：建议提前至少2小时预约，以确保有可用时段
3. **手机号**：确保提供正确的手机号，以便门店联系确认

### 注意事项
1. **预约时间格式**：必须使用ISO 8601格式（如：2024-01-15T14:00:00）
2. **人数限制**：单次预约人数最多为20人
3. **取消政策**：如需取消预约，请至少提前1小时联系门店
4. **数据更新**：门店和服务数据可能会定期更新，请以最新数据为准
