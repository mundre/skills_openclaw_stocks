# 万象绘卷

> 全能故事创作与交互体验工具箱

## 功能概览

- **56种文风切换**：从殿堂文学到网文爽文，一键切换
- **网文创作全流程**：黄金三章、细纲生成、平台适配
- **交互式故事模拟**：实时互动、角色扮演
- **异世界人生模拟**：回合制人生、代代相传、后台成书
- **剧情管理系统**：长篇故事管理、大总结系统
- **异世界人生模拟**：回合制人生体验、世界观随机生成、代代相传、自由行动AP系统、后台静默成书
- **拆书融合创作**：多源小说融合创作新作品
- **小说档案馆**：爬取、分析、存储小说素材

## ⚠️ 透明声明

### 爬虫脚本说明

本技能包含以下爬虫脚本（位于 `scripts/` 目录）：

| 脚本 | 目标站点 | 用途 |
|------|----------|------|
| `crawl_novel_index.py` | novel.transchinese.org | 爬取分类索引与下载txt |
| `crawl_jh_novel.py` | 刺猬猫、起点等多平台 | 聚合小说搜索下载 |
| `crawl_fq_novel.py` | fanqienovel.com | 番茄小说搜索下载 |

**重要声明**：
- 所有爬虫脚本**仅供学习参考使用**
- 用户需自行确认目标站点的许可与版权合规
- 爬取内容不得用于商业用途
- 建议限制爬虫范围与频率，遵守 robots.txt

**数据流向说明**：
- ✅ **仅下载**：脚本只从目标站点下载内容到本地
- ❌ **不上传**：脚本不会向任何外部端点发送数据
- 📁 **本地存储**：所有下载内容仅存储在用户本地设备
- 🔧 **手动运行**：脚本需要用户手动执行，不会自动运行

### 外部 API 声明

**关于爬虫脚本的说明**：
- 爬虫脚本执行的是**HTTP网页请求**，不是API调用
- API调用通常指调用第三方服务的编程接口（如OpenAI API、云服务等）
- 本技能**不包含任何第三方服务API调用**
- **无后端服务器、无云服务、无远程API**

**网络请求范围**：
- 爬虫脚本：仅用于从公开网站下载小说内容（用户手动运行）
- 无其他网络请求：技能核心功能不需要网络连接

### 凭据要求

- **无需任何环境变量或凭据**
- **无需 API 密钥**
- 所有功能均为本地运行

### 安全约束

- 本技能**不包含任何绕过模型安全限制的配置**
- 所有 `forbid_overrides`、`jailbreak`、`NSFW` 相关配置已移除

## 模块联动

所有模块不再独立，形成完整创作生态：

- **文风系统**（第二章）→ 交互故事/人生模拟/网文创作 的叙事风格引擎
- **创作引擎**（第五章）→ 人生模拟后台成书的去AI化处理 / 网文创作的质量保障
- **剧情管理**（第四章）→ 人生模拟的阶段大弧+连锁支线 / 交互故事的多线管理
- **交互式故事**（第三章）↔ 人生模拟（第十四章）两模式共享世界观+角色数据，可互相切换
- **拆书融合**（第十一章）→ 人生模拟的世界观素材来源 / 文风系统的参考库
- **小说档案馆**（第十三章）→ 人生模拟的成书存档 / 拆书分析的素材来源
- **质量控制**（第七章）→ 所有创作输出（网文/小说/后台成书）的审核把关
- **李诞七步**（第十五章）→ 人生模拟结局写作 / 网文开篇写作 的结构框架
