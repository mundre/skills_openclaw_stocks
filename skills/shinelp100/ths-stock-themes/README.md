# ths-stock-themes

获取同花顺个股题材/概念板块和人气排名数据。

## 功能特性

- 🏷️ **个股题材查询** — 获取股票所属的题材/概念板块
- 🔥 **人气排名榜单** — 获取同花顺个股人气排名
- 📊 **综合报告** — 生成题材 + 人气的 Markdown 报告
- 🌐 **Browser 自动化** — 直接从同花顺网站获取数据

## 使用方法

### 查询单只股票题材

```bash
python3 scripts/fetch_themes.py [股票代码]
```

示例：
```bash
python3 scripts/fetch_themes.py 000001
python3 scripts/fetch_themes.py 600519
```

### 获取人气排名榜单

```bash
python3 scripts/fetch_popularity.py --limit 20
```

### JSON 输出（程序处理）

```bash
python3 scripts/fetch_themes.py 000001 --json
python3 scripts/fetch_popularity.py --limit 20 --json
```

## 数据来源

- **个股详情**：`https://stockpage.10jqka.com.cn/[股票代码]/`
- **人气排名**：`https://www.iwencai.com/unifiedwap/result?w=个股人气排名`

## 注意事项

- 人气排名数据实时变化，建议注明数据获取时间
- 避免高频请求，脚本已添加适当延迟
