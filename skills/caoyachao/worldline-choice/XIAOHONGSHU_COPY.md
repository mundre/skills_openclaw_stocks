## 文案三：主打硬核挑战（技术向）

**标题**：
🎲 跑团玩家狂喜！这个AI游戏把d20检定玩明白了

**正文**：

作为DND老玩家，我被这个开源项目震惊了！🤯

它把桌面角色扮演的核心机制——d20检定——完美融入了AI叙事！

🎲 核心机制：
• d20 + 属性修正 ≥ DC（难度等级）
• 自然20必成功，自然1必失败
• 大成功/成功/勉强成功/失败/大失败，五个档位

这和跑团一模一样！

⚔️ 但更强的是——战术多步骤判定

你可以设计复杂的战术，比如：
```
1. 派100人偷袭粮草
2. 派500人埋伏两侧
3. 主力诈败诱敌
4. 合围击杀
```

每一步都要掷骰子！
而且步骤之间有依赖关系：
✅ 步骤1成功 → 步骤2难度降低
❌ 步骤1失败 → 步骤2难度飙升
❌ 关键步骤失败 → 整个计划中断

💡 还有防套利机制（v3.3.2更新）
之前可以"垫刀"蹭加成，现在不行了：
• 勉强成功只给DC-1（原来是-3）
• 步骤超过3个开始过载惩罚
• 大失败会泄露情报，后续更难

这设计太精妙了！奖励真正的策略，惩罚无脑堆步骤👏

🛠️ 技术栈：
• Python 3.8+
• 纯开源，MIT协议
• GitHub: caoyachao/worldline-choice

安装：
```
git clone https://github.com/caoyachao/worldline-choice.git
cd worldline-choice
python3 worldline_engine.py --new "三国" "谋士" "诸葛亮"
```

✔️ 策略游戏爱好者
✔️ 喜欢挑战的硬核玩家
