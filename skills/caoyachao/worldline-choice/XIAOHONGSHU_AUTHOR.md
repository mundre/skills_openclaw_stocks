## 主打文案：我发布了能写自己剧本的AI游戏

**标题**：
📖 我做了个AI游戏，朋友们玩到凌晨3点…每个选择都是真实的世界线

**正文**：

朋友们！我发布了一个AI互动叙事游戏！🎮

不是那种选A选B剧情都一样的假开放
是真的——你想做什么，游戏就给你什么回应！

🌟 这是啥？

Worldline Choice 是我开发的AI驱动叙事引擎
你可以扮演任何角色，在任何世界观里
做你想做的任何事

🎭 能玩什么？

**历史架空**：在三国当谋士、武将、甚至自立为王
**名著改编**：在西游改写天命，在水浒决定招安还是造反
**原创世界**：克苏鲁、蒸汽朋克、末日废土…
**甚至你自己的世界观**：和AI一起创造一个全新的世界

💡 核心体验：没有固定剧本

传统游戏：编剧写好了3个结局，你选一个
我的游戏：AI根据你的每一个选择，实时生成剧情
你的每个决定，都在创造一条全新的世界线

🎮 怎么玩？超简单！

**安装**（有手就会）：
```bash
git clone https://github.com/caoyachao/worldline-choice.git
cd worldline-choice
python3 worldline_engine.py
```

需要Python 3.8+，Mac/Windows/Linux都能跑

**开局**：
选择世界观 → 创建角色 → 开始你的故事

**进行游戏**：
每个场景都有4个选项 + 1个自由输入
不想选预设？直接打字告诉AI你想干嘛！

🔥 举个栗子（测试玩家的真实反馈）：

开局在颍川遇到李典
**选项A**：投靠曹操
**选项B**：投靠袁绍
**选项C**：保持中立
**选项D**：我另有打算…

玩家选了D，打字输入：
"我想试探一下李典，看他愿不愿意跟我一起干"

AI的回应：
"李典微微一愣，上下打量了你一眼：'阁下口气不小，凭什么让我跟你走？'"

然后系统**基于客观事实判定**——你的说服力、李典的忠诚度、当前局势，综合计算结果。
结果这位玩家运气不好，判定失败😭

李典直接拔刀："原来是个狂徒！" 
当场被追杀，第一局就这么凉了…

但！
第二把重开，同样的开局
换了个策略，判定成功
剧情完全不一样！
一起去投了刘备，后来居然在三足鼎立里活到了最后！

🎯 为啥上头？

✨ **真正的自由**
不限于四个选项，你可以输入任何想法
AI会判断可行性，给出对应的剧情

✨ **每次都不一样**
同样的开局，不同的选择 = 完全不同的故事
测试玩家玩了5遍三国篇，5次结局都不一样

✨ **失败也有趣**
被追杀、被背叛、战死沙场…
每种失败都有对应的剧情，而且往往比成功更精彩

✨ **你的选择被记住**
NPC会记得你之前做过什么
背叛过的人下次见你态度会不一样
这种"真实感"是我最想做的

⚔️ 关于那个战术推演…

游戏里还有个战术模式
比如你设计一个复杂的计划：
1. 派100人偷袭粮草
2. 派500人埋伏两侧
3. 主力诈败诱敌
4. 合围击杀

每一步都会**基于客观事实判定**——兵力对比、地形优势、将领能力、天气时机…
不是简单的成功/失败，而是综合各种因素实时计算结果

战术设计得太激进？可能执行力跟不上
太保守？可能错过战机
这需要真正的策略思维

💬 适合谁玩？

📚 爱看小说、想自己写故事的
🎭 喜欢角色扮演、代入感的
🧠 嫌普通游戏剧情太无聊的
⚔️ 想体验"军师""谋士"感觉的
🏛️ 对历史架空感兴趣的

这是你和AI一起写的一本小说
而你，就是主角📖✨
