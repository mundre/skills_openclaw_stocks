# Agent Learner

- **name**: Agent Learner
- **description**: AI self-learning from mistakes and corrections. Log errors, track patterns, generate improvement plans, and build best practices. Commands: mistake, correction, pattern, improve, quiz, report, reset, best-practices. Use when an AI agent needs to learn from failures and user feedback.

## Steps to Self-Improvement

1. **Log mistakes** — Record what went wrong and the context.
2. **Capture corrections** — When a user corrects you, log the before/after.
3. **Find patterns** — Identify recurring mistake categories.
4. **Build a plan** — Generate targeted improvement actions.
5. **Self-test** — Quiz yourself on past mistakes to reinforce learning.
6. **Track progress** — Review reports to measure improvement over time.
7. **Extract practices** — Distill learnings into reusable best practices.

## Commands

| Command | Purpose |
|---------|---------|
| `mistake` | Log a mistake |
| `correction` | Log user correction |
| `pattern` | Find recurring issues |
| `improve` | Generate improvement plan |
| `quiz` | Self-test on past mistakes |
| `report` | Progress report |
| `reset` | Start fresh |
| `best-practices` | Output learned practices |
